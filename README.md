# Deloitte Data Transformation Task

## 📌 Overview
This project focuses on transforming two different JSON data formats into a unified structure. It simulates a real-world scenario where data from multiple IoT devices needs to be standardized for analysis.

## 🎯 Objective
- Convert different input formats into a common output format
- Handle inconsistencies in data structure
- Normalize timestamps and device information

## 🛠️ Technologies Used
- Python
- JSON

## 🔧 Key Features
- Handles multiple JSON formats dynamically
- Converts timestamps into milliseconds
- Extracts and standardizes device details
- Normalizes location data
- Automatically detects status and temperature fields

## ▶️ How to Run
1. Clone the repository
2. Place `data-1.json` and `data-2.json` in the same folder
3. Run:
   ```bash
   python main.py
